Урок 1.1: Инфраструктура

Курс разработки веб-приложений (Laravel Junior)

1. Как работает интернет?

Веб работает по модели Клиент-Сервер:

  • Браузер (Клиент): Это программа (Chrome, Firefox), которая отправляет запрос.
  • Сервер: Это мощный компьютер, который хранит файлы сайта и базу данных.

Когда вы вводите адрес сайта, браузер просит сервер: "Дай мне файл index.html". Сервер отправляет код, а браузер превращает этот код в красивую картинку.

2. Инструментарий разработчика

Для работы нам понадобится VS Code. Чтобы верстать быстрее, установим плагины:

  • Live Server: Позволяет видеть изменения на сайте мгновенно после сохранения файла.
  • Auto Rename Tag: Автоматически меняет закрывающий тег, если вы изменили открывающий.

3. Структура HTML-файла

Каждая страница начинается с базового "скелета". Скопируй этот код в свой файл index.html:

<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Моя страница</title>
</head>
<body>
    <h1>Привет, школьники!</h1>
    <p>Мы начинаем путь в веб-разработку.</p>
</body>
</html>
Что значат эти теги?
  • <!DOCTYPE html> — сообщает браузеру, что это современный стандарт HTML5.
  • <html> — самый главный контейнер для всей страницы.
  • <head> — здесь хранятся настройки (кодировка, заголовок вкладки).
  • <body> — здесь живет всё, что мы видим на экране.

4. Практические задания

Задание №1: Первая страница

Создай файл index.html, набери структуру вручную и добавь заголовок <h1> со своим именем.

Задание №2: Работа с Live Server

Запусти файл через Live Server. Измени текст в редакторе и нажми Ctrl + S. Убедись, что в браузере всё обновилось само.

Задание №3: Творчество

Добавь в <body> три разных абзаца <p>, в которых напиши свои любимые игры или хобби.
